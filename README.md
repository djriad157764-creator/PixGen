# ✨ PixGen - AI Photo Generator

![Banner](/public/banner.png)

> **Turn Ideas into Stunning AI Art**

## 📖 Project Overview

**Project Type:** AI Photo Generator Web Application

**PixGen** is a modern, full-stack web application that empowers users to turn their creative ideas into stunning AI-generated art. With an intuitive and premium interface, users can generate high-quality images from simple text prompts, browse global top generations, and manage their personal creative portfolios. 

---

## 🚀 Technologies Used

This project leverages a modern, cutting-edge tech stack:

*   **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
*   **Library:** [React 19](https://react.dev/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **UI Components:** [HeroUI](https://heroui.com/) & [@iconify/react](https://iconify.design/)
*   **Authentication:** [Better-Auth](https://better-auth.com/)
*   **Database:** [MongoDB](https://www.mongodb.com/)
*   **Notifications:** [React Hot Toast](https://react-hot-toast.com/)

---

## ✨ Features

*   **🎨 AI Image Generation:** Generate stunning images instantly using text prompts.
*   **🖼️ Public Gallery:** Browse and discover top generations and all photos created by the community.
*   **🔐 Secure Authentication:** Seamless Sign Up and Sign In functionality powered by Better-Auth.
*   **👤 User Profiles:** Personalized dashboards to view and manage generated artworks.
*   **💳 Pricing Plans:** Tiered subscription structures for different generation limits.
*   **📱 Fully Responsive:** A highly responsive design providing a seamless experience across desktop, tablet, and mobile devices.
*   **💀 Skeleton Loaders:** Smooth loading states for better UX.

---

## 🧩 Sections & Components

*   **Landing Page:** Engaging hero banner and a showcase of top AI generations.
*   **Authentication:** Dedicated, secure pages for user registration and login.
*   **Pricing:** Clear and concise subscription options.
*   **Profile:** A private space for users to manage their creations.
*   **Gallery:** A masonry or grid layout showcasing community art.
